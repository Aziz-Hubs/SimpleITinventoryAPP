# Project Knowledge Base

## Overview

This is a Simple IT Inventory Management application built with Next.js and styled with Tailwind CSS.

## Key Features

- Inventory Tracking
- Employee Assignment
- Maintenance Lifecycle
- Hardware Models
- Deployment Operations

## Architecture

- **Frontend**: Next.js 14 (App Router)
- **State**: React Hooks + LocalStorage (Mock)
- **API**: Centralized service layer

## Development

- `npm run dev` to start
- `npm run build` to production
